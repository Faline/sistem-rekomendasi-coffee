# sistem-rekomendasi-coffee

## Deskripsi
Proyek ini merupakan implementasi Sistem Rekomendasi Produk Kopi berbasis metode Hybrid Recommendation System yang menggabungkan pendekatan Content-Based Filtering (CBF) dan Collaborative Filtering (CF).

Sistem ini bertujuan untuk memberikan rekomendasi produk kopi yang lebih personalized, dengan mempertimbangkan preferensi pengguna, karakteristik produk, dan pola interaksi pembelian.

## Tujuan
- Mengatasi masalah cold start pada sistem rekomendasi
- Meningkatkan akurasi dan relevansi rekomendasi
- Memberikan pengalaman pengguna yang lebih personal

## Cara Menjalankan Project
1. Clone Repository
   ```bash
   git clone https://github.com/username/repo-name.git
   cd repo-name
3. Jalankan Backend (Flask)
   ```bash
   pip install -r requirements.txt
   python app.py
5. Jalankan Frontend (Laravel)
   ```bash
   cd ui
   composer install
   php artisan serve

## Hasil dan Manfaat
- Rekomendasi lebih relevan dan personal
- Mengurangi masalah cold start
- Meningkatkan pengalaman pengguna dalam memilih produk kopi
